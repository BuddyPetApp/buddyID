# Fotos de fundo por pergunta (BuddyID Flow)

Cada pergunta do formulário tem uma foto de fundo (full-bleed). Para trocar,
**substitui o ficheiro mantendo o mesmo nome** (`.jpg`). Não é preciso mexer no código.

Os ficheiros atuais são **placeholders** (cópia de `advisor-francisco.jpg`) — substitui por
fotos de cães reais, fofinhos e, se possível, relacionados com cada pergunta.

Recomendações:
- Orientação vertical/retrato (o lado esquerdo no desktop é alto e estreito).
- ~1080×1600 px, `.jpg`, idealmente < 400 KB cada.
- O cão/foco no centro (em cima e em baixo há um leve escurecimento para o texto branco ler bem).

## Mapa ficheiro → pergunta (ordem do fluxo)

| Ficheiro          | Pergunta no ecrã                              | Sugestão de foto                          |
|-------------------|-----------------------------------------------|-------------------------------------------|
| `q1.jpg`          | Como se chama o teu cão?                       | Cão fofo a olhar para a câmara            |
| `q2.jpg`          | Qual é a raça do teu cão?                      | Cão de corpo inteiro / raça distinta      |
| `q3.jpg`          | Idade e género                                 | Cachorro + cão adulto, ou cão expressivo  |
| `q4.jpg`          | Comportamento do cão                           | Cão a brincar / a interagir               |
| `qOwner.jpg`      | Autoavaliação do Tutor                         | Tutor com o cão ao colo / a passear       |
| `q5.jpg`          | Na trela, o teu cão...                         | Cão à trela num passeio                   |
| `q6.jpg`          | Habitação e Alimentação                        | Cão em casa / junto à taça                |
| `q7b.jpg`         | Quem mais vive em casa?                        | Cão com família / outros animais          |
| `q8.jpg`          | Rotina e Origem                                | Cão a correr ao ar livre                  |
| `qLocation.jpg`   | Onde moras?                                     | Cão na cidade / parque                    |
| `q12.jpg`         | O teu cão tem medos ou fobias?                 | Cão tímido / aconchegado                  |
| `q13.jpg`         | Que serviços normalmente procuras?            | Cão no veterinário / banho / creche       |
| `q14.jpg`         | O que mais queres para o teu cão?             | Cão feliz / socialização                  |
| `qConcern.jpg`    | Há alguma coisa que te preocupe no teu cão?   | Cão calmo / olhar atento                  |
| `consent.jpg`     | Antes de terminar (criar conta)               | Cão celebração / muito fofo               |
